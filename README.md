# Shell Shock: Fire Mission
\
_previous: "Old-Weapon-Simulator"_

\
A short story-driven game about setting up an artillery piece and completing missions.  
You play as a droid named ___JSH-V1___, not knowing how you ended up on an artillery piece. 
Only you can decide which path to take:

* complete missions and improve your reputation with the boss.
* or go against the rules and face the consequences.
